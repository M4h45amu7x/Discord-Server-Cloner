## Description:

Clone the Discord server such as icon, emojis, roles, categories, channels, permissions

## Features:

<ul>
  <li>Clone guild icon</li>
  <li>Clone emojis</li>
  <li>Clone roles</li>
  <li>Clone categories</li>
  <li>Clone channels</li>
  <li>Clone permissions</li>
</ul>

## Usages:

```javascript
npm install && npm run build && npm run start
```
